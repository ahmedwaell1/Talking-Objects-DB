# Talking Objects Database

## How to use
1. Create empty database (GRADTEST)
2. Run scripts in order:

database/schema/01_schema.sql  
database/seeds/02_seed_master.sql  
database/seeds/03_seed_objects.sql  
database/seeds/04_seed_content.sql  
database/seeds/05_seed_sessions.sql  
database/seeds/06_seed_search.sql  

## Notes
- Do NOT upload MDF or backups
- Database is reproducible from scripts
